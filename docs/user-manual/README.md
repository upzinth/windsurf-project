## การเริ่มต้นใช้งาน

### ข้อกำหนดเบื้องต้น
- เว็บเบราว์เซอร์ที่รองรับ: Chrome, Firefox, Safari, Edge (เวอร์ชันล่าสุด)
- การเชื่อมต่ออินเทอร์เน็ตที่เสถียร
- อีเมลสำหรับการยืนยันตัวตนและรับการแจ้งเตือน

### การเข้าถึงระบบ
1. เปิดเว็บเบราว์เซอร์
2. ไปที่ URL: https://9tools.upz.in.th
3. คลิกที่ปุ่ม "เริ่มต้นใช้งาน" หรือ "เข้าสู่ระบบ"

## การล็อกอินและการยืนยันตัวตน

### การล็อกอินด้วย Gmail
1. คลิกที่ปุ่ม "เข้าสู่ระบบด้วย Gmail"
2. เลือกบัญชี Gmail ที่ต้องการใช้
3. อนุญาตการเข้าถึงข้อมูลที่จำเป็น
4. ระบบจะนำคุณไปยังหน้าหลักของระบบ

### การล็อกอินด้วยอีเมลและรหัสผ่าน
1. ป้อนอีเมลและรหัสผ่านในช่องที่กำหนด
2. คลิกที่ปุ่ม "เข้าสู่ระบบ"
3. หากเปิดใช้งาน 2FA ให้ป้อนรหัส OTP ที่ได้รับทางอีเมล

### การตั้งค่า Two-Factor Authentication (2FA)
1. ไปที่เมนู "โปรไฟล์" > "การตั้งค่าความปลอดภัย"
2. เปิดใช้งาน "Two-Factor Authentication"
3. บันทึกรหัส backup codes ที่แสดง
4. สแกน QR Code ด้วยแอปพลิเคชัน Authenticator

### การรีเซ็ตรหัสผ่าน
1. คลิกที่ "ลืมรหัสผ่าน?"
2. ป้อนอีเมลที่ลงทะเบียนไว้
3. ตรวจสอบอีเมลสำหรับลิงก์รีเซ็ตรหัสผ่าน
4. ตั้งรหัสผ่านใหม่และยืนยัน

## การจัดการผู้ใช้งาน

### ระดับผู้ใช้งาน

#### แอดมิน (Admin)
- จัดการผู้ใช้งานทั้งหมด
- ตั้งค่าระบบและนโยบายความปลอดภัย
- ดูประวัติการใช้งานทั้งหมด
- สร้างรายงานและสถิติ
- จัดการความจุและโควต้า

#### แมเนเจอร์ (Manager)
- ดูและดาวน์โหลดไฟล์ทั้งหมด
- อัปโหลดเอกสาร PDF และไฟล์อื่นๆ
- จัดการเอกสารในโฟลเดอร์
- สร้างและจัดการหมวดหมู่
- ดูประวัติการใช้งานของผู้ใช้ในระดับตนเอง

#### ยูเซอร์ (User)
- ดูรายละเอียดไฟล์ PDF
- ดาวน์โหลดไฟล์ที่ได้รับอนุญาต
- ค้นหาเอกสารที่เข้าถึงได้
- ดูประวัติการดาวน์โหลดของตนเอง

## การจัดการเอกสาร

### การสร้างโฟลเดอร์
1. คลิกที่ปุ่ม "สร้างโฟลเดอร์ใหม่"
2. ป้อนชื่อโฟลเดอร์และคำอธิบาย (ถ้าต้องการ)
3. เลือกโฟลเดอร์หลัก (ถ้าเป็นโฟลเดอร์ย่อย)
4. ตั้งค่าสิทธิ์การเข้าถึง
5. คลิก "บันทึก"

### การอัปโหลดไฟล์
1. ไปที่โฟลเดอร์ที่ต้องการอัปโหลด
2. คลิกที่ปุ่ม "อัปโหลดไฟล์" หรือลากไฟล์ไปวาง
3. เลือกไฟล์ที่ต้องการอัปโหลด
4. กรอกข้อมูลเอกสาร (ชื่อ, คำอธิบาย, หมวดหมู่)
5. ตั้งค่าสิทธิ์การเข้าถึง
6. คลิก "อัปโหลด"

### การจัดการไฟล์
- **ดูไฟล์**: ดับเบิลคลิกที่ไฟล์เพื่อเปิดดู
- **ดาวน์โหลด**: คลิกขวาและเลือก "ดาวน์โหลด"
- **แก้ไขข้อมูล**: คลิกที่ไอคอนแก้ไขเพื่ออัปเดตข้อมูล
- **ย้ายไฟล์**: ลากไฟล์ไปยังโฟลเดอร์อื่น
- **ลบไฟล์**: คลิกที่ไอคอนถังขยะและยืนยัน

### การแปลงไฟล์เป็น PDF
ระบบจะแปลงไฟล์เป็น PDF อัตโนมัติเมื่อ:
- อัปโหลดไฟล์ที่รองรับ (Word, Excel, PowerPoint, รูปภาพ)
- ไฟล์ PDF ใหม่จะปรากฏในโฟลเดอร์เดียวกัน

## การค้นหาเอกสาร

### การค้นหาขั้นพื้นฐาน
1. ใช้ช่องค้นหาที่ด้านบนของหน้า
2. ป้อนคำค้นหา (ชื่อไฟล์ หรือคำในเนื้อหา)
3. กด Enter หรือคลิกไอคอนค้นหา

### การค้นหาขั้นสูง
1. คลิกที่ "ค้นหาขั้นสูง"
2. กรองตาม:
   - ประเภทเอกสาร
   - วันที่อัปโหลด
   - ผู้อัปโหลด
   - ขนาดไฟล์
   - แท็ก
3. คลิก "ค้นหา"

### การใช้แท็ก
- แท็กช่วยจำแนกเอกสาร
- คลิกที่แท็กเพื่อดูเอกสารที่เกี่ยวข้อง
- สามารถค้นหาโดยใช้หลายแท็กพร้อมกัน

## ระบบแจ้งเตือน

### การแจ้งเตือนในระบบ
- ปรากฏที่มุมขวาบนของหน้า
- แสดงการแจ้งเตือนล่าสุด
- คลิกเพื่อดูรายละเอียด

### การแจ้งเตือนทางอีเมล
- ได้รับเมื่อมีการอัปโหลดไฟล์ใหม่
- แจ้งเตือนเมื่อไฟล์ถูกดาวน์โหลด
- เตือนเมื่อเอกสารใกล้หมดอายุ

### การจัดการการแจ้งเตือน
1. ไปที่ "การตั้งค่า" > "การแจ้งเตือน"
2. เลือกประเภทการแจ้งเตือนที่ต้องการรับ
3. ตั้งค่าวิธีการรับ (ในระบบ, อีเมล, ทั้งสองอย่าง)

## ความปลอดภัยและความเป็นส่วนตัว

### การเข้ารหัสไฟล์
- ไฟล์ทั้งหมดถูกเข้ารหัสด้วย AES-256
- การส่งข้อมูลผ่าน HTTPS เท่านั้น
- การเข้าถึงถูกตรวจสอบทุกครั้ง

### การควบคุมการเข้าถึง
- สิทธิ์การเข้าถึงระดับโฟลเดอร์และไฟล์
- บันทึกการเข้าถึงทุกรายการ
- การจำกัดจำนวนครั้งการล็อกอินผิดพลาด

### ประวัติการใช้งาน (Audit Trail)
- บันทึกทุกการกระทำในระบบ
- ตรวจสอบได้ว่าใครทำอะไรเมื่อไหร่
- ส่งออกรายงานได้

## การแก้ไขปัญหา

### ปัญหาการล็อกอิน
- **ลืมรหัสผ่าน**: ใช้ฟังก์ชัน "ลืมรหัสผ่าน"
- **บัญชีถูกล็อก**: รอ 15 นาทีหรือติดต่อแอดมิน
- **OTP ไม่ทำงาน**: ตรวจสอบเวลาอุปกรณ์และ backup codes

### ปัญหาการอัปโหลดไฟล์
- **ไฟล์ใหญ่เกินไป**: แบ่งไฟล์หรือติดต่อแอดมิน
- **ประเภทไฟล์ไม่รองรับ**: ตรวจสอบประเภทไฟล์ที่อนุญาต
- **พื้นที่ไม่เพียงพอ**: ลบไฟล์ที่ไม่จำเป็นหรือขอเพิ่มพื้นที่

### ปัญหาความเร็ว
- **โหลดช้า**: ตรวจสอบความเร็วอินเทอร์เน็ต
- **ดาวน์โหลดช้า**: ลองดาวน์โหลดในเวลาที่ใช้งานน้อย
- **ค้นหาช้า**: ใช้ตัวกรองเพื่อจำกัดผลลัพธ์
